Fonts - Web

Shown below are all acceptable fonts for DID(it) and DID(it) Applications in the Rails Framework. Within the DID(it) framework there are three fonts to choose from to accommodate different styles and brandings within the themes. Offering Perpetua as a serif font, with two San Serif fonts, one standard and one condensed.

All other font guidelines outlined in print should then be consistent on the web – Avoid using Perpetua as body copy to avoid pixel distortion, use no more than two font styles within a site (gives the feeling of order and consistency). Mixing san serif and serif are fine.

All DID(it) branding communications on the web should follow these same guidelines with one exception: All titles should be Perpetua to stay consistent with the logo.
