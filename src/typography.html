Typography

Typography is an important part of branding in making sure the message is consistent. All DID(it) communications should stay consistent with the logo, using Perpetua as all titles and subtitles where possible. Verdana should be used for all body text and can be used for subtitles when it’s found to work better in a document. Avoid using Perpetua in body copy because of the thin lines which can pixelate when smaller. New Times Roman can be substituted where Perpetua isn’t available.

DID(it) prefers Verdana as it’s sans-serif because the round open format of the lettering allows for better viewing on smaller devices, and is clearer for those with assistive technology. Open Sans may substituted where Verdana isn’t available
