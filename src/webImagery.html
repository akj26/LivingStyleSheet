Web Imagery

The DID(it) style for images used in print and web are two styles:

1. Consistent with USCIS-DHS – colorize photographs with the Homeland Security Blue tint with 1 pt light grey stroke. On web within themes images can be colorized to go with theme colors.

2. Full color and cropped close

3. Should be clearly relevant to subject matter. Avoid standard clip art. Use DHS library as a resource where possible and check license information. Patriotic, American, technology, modern, simple, clean should be key words when choosing an image


A colored pictured de-saturated, and then colorized in Photoshop using the theme colors for consistency. It is recommended that you use only one color per theme for tinting images.
