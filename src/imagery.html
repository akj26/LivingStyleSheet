Imagery

The DID(it) style for images used in print and web are two styles:

1. Consistent with USCIS-DHS – colorize photographs with the Homeland Security Blue tint with 1 pt light grey stroke.

2. Full color and cropped close, has color coordination in the image (for example the DHS blue mouse below).

3. Should be clearly relevant to subject matter. Avoid standard clip art. Use DHS library as a resource where possible and check license information. Patriotic, American, technology, modern, simple, clean should be key words when choosing an image.
